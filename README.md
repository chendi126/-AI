# 🐾 傻毛宠物（Shamao Pets）

> 一个专为养宠人打造的「知识 + 社区 + AI」移动应用。

---

## 简介

**傻毛宠物**是一款面向所有养宠人士的垂直社区应用。

养宠人每天都在面对各种具体问题：该喂什么粮、身上长了东西是什么病、怎么训练上厕所、哪家宠物医院靠谱……但市场上至今没有一款真正**专门**为宠物设计的产品。用户只能在百度搜索、小红书翻帖、问通用 AI，在混杂的信息里大海捞针。

傻毛宠物想做的是：**一个只有养宠人、只解决养宠问题的专属空间。**

---

## ✨ 核心功能

| 模块 | 功能说明 | 状态 |
|------|----------|------|
| **品种百科** | 结构化品种档案（饮食/健康/护理/社区），覆盖猫、狗、异宠 | 🚧 MVP |
| **内容社区** | 双列瀑布流，支持图文/视频发布，按品种标签精准分发 | 🚧 MVP |
| **傻毛 AI** | 基于大模型的宠物知识问答助手，快速获取精准答案 | 🚧 MVP |
| **宠物商城** | 按品种精准推荐商品，社区真实评价辅助决策 | 📋 规划 |
| **宠物档案** | 集中管理疫苗、驱虫、体检记录，智能到期提醒 | 📋 规划 |
| **品种匹配** | 通过测试帮助潜在养宠人找到适合的品种 | 📋 规划 |
| **领养服务** | 对接救助机构，提供正规领养渠道 | 📋 规划 |

## 🚀 快速开始

本项目是一个 **纯前端 MVP 原型**，无需构建工具，直接浏览器打开即可预览。

```bash
# 1. 进入项目目录
cd 宠物app

# 2. 用浏览器打开 index.html
# 或启动一个本地服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

**预览建议**：在 Chrome/Edge 开发者工具中切换至 **移动端视口**（375px 宽度），以获得最佳体验。

## 🎯 产品愿景

> **汇集全球养宠人，分享经验、故事、心得。**

宠物寿命比人短，这注定是一场带着倒计时印记的陪伴。我们希望：
- 新手不再焦虑，一站式获取可信的养宠知识；
- 养宠人找到同好，在社区里被真正理解；
- 当那一天到来，有一个安静的地方可以悼念。
